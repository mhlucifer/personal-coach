# AI Agent Roadmap

This route is secondary. Use it to improve personal workflow, not to escape BIOS execution.

## Level 0: Prompted Workflow

- Turn repeatable learning tasks into stable prompts.
- Keep outputs small and updateable.

## Level 1: Memory And Retrieval

- Keep source index, logs, weakness, and reusable exams.
- Retrieve local notes before asking for new explanation.

## Level 2: Tooling

- Use local search, file inspection, and structured templates.
- Prefer evidence from files over free-form summaries.

## Level 3: Automation

- Only automate a workflow after it has worked manually for several sessions.
