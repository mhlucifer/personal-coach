# Code Reading Loop

Use this when reading BIOS code.

## Steps

1. Find the entry function.
2. Find key structs.
3. Find data sources.
4. Find branch conditions.
5. Find output targets.
6. Find verification commands or observable logs.
7. Find modification risks.
8. Generate a call-chain and data-flow summary.

## Required Output

- Entry function.
- Call chain.
- Input source.
- Output target.
- Static path.
- Dynamic path.
- Debug logs to add.
- Risks if modified.
- One-paragraph summary in my own words.
