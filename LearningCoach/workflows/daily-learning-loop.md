# Daily Learning Loop

Target time: 45-60 minutes.

## 1. Locate: 5 Minutes

- What is today's real problem?
- Which roadmap and level does it belong to?
- Which one smallest point will be solved today?

## 2. Learn: 25 Minutes

- Use BIOS Code Tutor for code problems.
- Use Foundation Patch Tutor only for a revealed foundation gap.
- Learn one point with real code or a real work scenario.

## 3. Output: 10 Minutes

Write a short explanation in my own words.

If I cannot explain it, the session is not finished. Reduce the scope and explain a smaller point.

## 4. Exam: 10 Minutes

Examiner asks five questions:

- concept
- code trace
- modification risk
- debug investigation
- counterexample or boundary condition

## 5. Persist: 5 Minutes

Memory Keeper updates:

- `logs/daily-log.md`
- `logs/weakness.md`
- `logs/questions.md`
- relevant notes or exam bank
