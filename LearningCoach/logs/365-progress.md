## Zero-Day Rule

If one `0` appears:

- Record the trigger.
- Next day must be `C` or better.

If two consecutive `0` days appear:

- Run `workflows/drift-recovery-loop.md`.
- Reduce the next day to the recovery plan.
- Block games until the recovery session is complete.

If three `0` days appear in one week:

- The next week is not a challenge week.
- Only the floor line matters.
- Remove one optional learning branch for seven days.
