# BIOS_AI 阶段汇报资料包

本目录用于中午在另一台机器或另一个 Codex 窗口继续修改 PPT。

## 包含内容

- `05_BIOS_AI_阶段汇报_瑞士工程风.pptx`  
  当前可编辑 PowerPoint 文件。
- `05_BIOS_AI_阶段汇报_瑞士工程风_contact_sheet.png`  
  12 页缩略总览，用于快速判断整体节奏。
- `01_BIOS_AI_阶段汇报_当前版大纲.md`  
  根据当前 PPT 反写的新大纲，后续改 PPT 优先看这个。
- `02_使用的_skills_和生成说明.md`  
  记录本次使用的 Presentations skill、风格规则、生成与 QA 方式。
- `03_文件清单.md`  
  文件用途和建议修改顺序。
- `deck_source.mjs`  
  本次 PPT 的 artifact-tool 源文件，后续需要程序化改版时可用。

## 继续修改建议

优先打开 `01_BIOS_AI_阶段汇报_当前版大纲.md`，确认每页结论和内容；再打开 PPTX 做局部文字、截图替换和排版微调。

当前版本已经预留了 AI Review 两处截图位，可以直接把对应证据截图贴进去。
